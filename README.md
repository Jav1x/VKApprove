# VKApprove
***
Скрип автоматически принимает все заявки на вступление в закрытую группу VK по указанному интервалу

### Использование:
1. Установить зависимости
```
pip install -r requirements.txt
```
2. В **settings.py** заменить параметры на свои
3. Запустить скрипт!
```
python main.py
```
